8 Steps to Create a Webpage

  1. Open Visual Studio Code on your computer. If you need to download it click here.

  2. Choose a directory on your computer where you would like to save your websites files. Something like "MyFirstWebsite" and make an "index.html" file inside of it.

  3. Code away! If you are unfamiliar with programing The Odin Project and MDN have great web development tutorials for beginners.

  4. Create a GitHub account if you do not alreay have one.

  5. Choose to create a new repository at the top right corner.


  6. Name your repository "(Username).github.io" and create it.

    *If you would like others to be able to view your page make sure its set to "Public".


  7. Choose to upload a new file.

    Then simply drag and drop all of your files for your website into your new repository. Be sure to hit "commit changes" at the bottom of the page.



  8. Celebrate!! You just sucessfully built your very own webpage!
    You may view it by typing your link into your browser (Username).GitHub.io
