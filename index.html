How To Create a Webpage

  1. Open Visual Studio Code on your computer. If you need to download it click here
  2. Choose a directory on your computer where you would like to save your websites files.
  3. Code away! If you are unfamiliar with programing The Odin Project and MDN have great web development tutorials for beginers.
  4. Create a GitHub account if you do not alreay have one
  5. Choose to create a new repository at the top right corner
  6. Name your repository "(Username).github.io" and create it.

    *If you would like others to be able to view your page make sure its set to "Public"
