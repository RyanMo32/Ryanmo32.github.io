How To Create a Webpage

  1. Download or open Visual Studio Code on your computer
  2. Choose a directory on your computer where you would like to save your websites files.
  3. Create a GitHub account if you do not alreay have one
  4. Choose to create a new repository at the top right corner
  5. Name your repository "(Username).github.io" and create it.

    *If you would like others to be able to view your page make sure its set to "Public"
